# 🎬 React Native Movie App 

This is my first fully functional cross-platform mobile application, built to explore the power of React Native and modern mobile development ecosystems. It features a beautiful UI, seamless file-based routing, and real-time backend integration.

## 🚀 Tech Stack Used:
- **Frontend Framework:** React Native with Expo
- **Styling:** NativeWind (Tailwind CSS for React Native)
- **Routing:** Expo Router (File-based routing)
- **Backend/Metrics:** Appwrite (BaaS)
- **Data Source:** TMDB API (The Movie Database)
